# Northwind SQL Analysis (SQL Server / SSMS)

## Overview
This project analyzes the classic Northwind dataset using SQL Server.
The focus is on sales performance, customer behavior, and order fulfillment metrics.

## Tools Used
- SQL Server (Developer Edition)
- SQL Server Management Studio (SSMS)
- VS Code (MSSQL Extension)

## Key Analyses
- Dataset overview and key metrics
- Revenue analysis by product
- Customer order behavior
- Order processing time and SLA analysis
- Customer ranking using window functions

## SQL Concepts Used
- JOINs
- GROUP BY / HAVING
- Date calculations (DATEDIFF)
- Window functions (RANK)
